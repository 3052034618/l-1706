# 回声工坊 - Echo Workshop

多人在线魔法世界游戏系统，玩家创建回声工坊，招募回声师，制作声波探测器，参加回声大赛。

## 功能特性

- 🏭 **回声工坊** - 创建工坊，招募回声师，管理成员
- 🔧 **探测器制作** - 收集材料，按配方制作，品质与词缀系统
- 🏆 **回声大赛** - 每日比赛，实时评分，技能对决
- 💰 **交易市场** - 材料与成品交易，价格智能建议
- 🏰 **公会系统** - 联合工坊，共鸣塔升级，全员增益
- 📊 **产业报告** - 数据统计，PDF导出
- 🏅 **排行榜** - 全服排名，收藏度、积分、贡献排行

## 技术栈

**后端:**
- Node.js + Express
- Socket.io (实时通信)
- Redis (缓存与消息队列)
- SQLite/PostgreSQL (数据存储)

**前端:**
- React 18 + TypeScript
- Redux Toolkit (状态管理)
- Socket.io Client
- Chart.js (数据可视化)
- jsPDF (PDF导出)

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

访问 http://localhost:3000 查看应用。
